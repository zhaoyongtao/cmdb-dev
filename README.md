## 安装部署

1. 通过pip进行需求环境的安装

   ```
   pip install -r requirement.txt
   ``` 
   
2. 启动测试Web服务

   ```
   python manage.py runserver
   ```
   
3. 浏览器

   ```
   http://127.0.0.1:8000
   ```
